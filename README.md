# Análisis de Transacciones Financieras

## Descripción del Proyecto

Este proyecto analiza datos de transacciones financieras utilizando Python, SQL y Power BI.

El objetivo principal fue limpiar, transformar y visualizar los datos para identificar patrones de gasto, comportamiento de clientes y métodos de transacción mediante un dashboard interactivo.

---

## Herramientas Utilizadas

* Python
* Pandas
* SQL (MySQL)
* Power BI
* Git & GitHub

---

## Preparación de Datos

Durante el proceso de limpieza y transformación se realizaron:

* Conversión y limpieza de importes
* Transformación de fechas
* Eliminación de valores nulos
* Clasificación de categorías MCC
* Creación de variables calculadas

---

## Dashboard

El dashboard incluye:

* KPIs financieros
* Análisis de gastos por categoría
* Clientes con mayor gasto
* Distribución de métodos de transacción
* Filtros interactivos

El dashboard fue diseñado para responder las siguientes preguntas:

* ¿Cómo se distribuyen los gastos entre las diferentes categorías?
    grafico categorias
* ¿Qué clientes presentan un mayor nivel de gasto?
    Top clients
* ¿Qué métodos de transacción son los más utilizados?
    donut chart
* ¿Cuál es el balance financiero general de las transacciones analizadas?
    KPIs
